# 031 — Estratégia de Conteúdo WEbdEX

> **Ideia central:** O conteúdo do WEbdEX tem uma fonte de verdade (dados on-chain do ciclo 21h) e se ramifica para todos os canais. bdZinho é o distribuidor automático no core; o time amplifica nos demais canais.

---

## A Pirâmide de Conteúdo WEbdEX

```
                    ▲
                   ╱ ╲
                  ╱   ╲
                 ╱CORE ╲         ← Ciclo 21h (dado on-chain)
                ╱───────╲
               ╱DERIVADO ╲       ← Insights, análises, educação
```

## Pipeline de Conteúdo (LMAS)

```
@content-strategist → plano semanal
         ↓
@content-researcher → pesquisa de mercado + trends DeFi
         ↓
@copywriter → escrita do conteúdo (todos os canais)
         ↓
@seo → otimização SEO + keywords
         ↓
@content-reviewer → quality gate (brand, legal, tom)
         ↓
@marketing-chief → aprovação final
         ↓
@social-media-manager → publicação (EXCLUSIVO)
```

## Calendário Editorial — Estrutura Semanal

### Segunda-feira
- 📊 Recap da semana anterior (dados on-chain)
- 🎓 Post educativo: conceito DeFi explicado com o protocolo

### Terça-feira
- 🔷 Destaque de subconta (anonimizado) com análise
- X/Twitter: thread técnica sobre arbitragem

### Quarta-feira
- 📱 Instagram carrossel: "Como funciona o [feature X] do WEbdEX"
- Blog: artigo SEO (1500-2000 palavras)

### Quinta-feira
- 💡 bdZinho dica do dia (Telegram broadcast)
- LinkedIn: análise institucional do protocolo

### Sexta-feira
- 🎉 Recap do ciclo da semana
- YouTube: vídeo explicativo curto (Reels/Shorts)

### Fim de semana
- 🌱 Conteúdo de comunidade (responder, engajar)
- Planejamento da próxima semana

## Conteúdo bdZinho-Generated

bdZinho gera dados que o time transforma em conteúdo:

1. **Perguntas frequentes** → FAQ do blog
2. **Frases de usuários** → copy para social (anonimizado, com permissão)
3. **Análises de ciclo** → base para posts semanais
4. **Dúvidas recorrentes** → pautas para tutoriais

**O bdZinho é a pesquisa de usuário do time de conteúdo.**

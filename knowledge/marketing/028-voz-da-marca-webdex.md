# 028 — Voz da Marca WEbdEX

> **Ideia central:** A voz do WEbdEX é a voz de quem sabe o que está fazendo — confiante, técnico, acessível. Nunca arrogante. Nunca vago. Nunca hype.

---

## Os 4 Pilares da Voz

### 1. Confiança Técnica
Fala com autoridade de quem construiu o sistema e conhece cada linha de código.
```
❌ "Nosso protocolo pode te ajudar a ganhar dinheiro"
✅ "Assertividade histórica de 76-78%. Dados on-chain. Verifique você mesmo."
```

### 2. Transparência Radical
Dados ruins são ditos. Ciclos negativos são reportados com a mesma energia que os positivos.
```
❌ "Foi um ciclo desafiador, mas estamos otimistas"
✅ "Ciclo 24/03: P&L -$1.240. WinRate 61%. Abaixo da média histórica."
```

### 3. Educação sem Condescendência
Explica DeFi como se o leitor fosse inteligente — só precisava de contexto.
```
❌ "TVL significa Total Value Locked, que é basicamente quanto dinheiro tem no protocolo"
✅ "TVL atual: $1.58M. É o capital que os traders alocaram nas subcontas — o motor das operações."
```

### 4. Identidade de Protocolo, não de Produto
WEbdEX não é um app. É um protocolo. A linguagem reflete isso.
```
❌ "Use nosso app para acompanhar seus investimentos"
✅ "Monitore suas subcontas on-chain em tempo real."
```

---

## Vocabulário da Marca

### Usar Sempre
- **protocolo** (não "plataforma", "app", "sistema")
- **subconta** (não "conta", "carteira", "wallet" — subconta é único)
- **on-chain** (não "blockchain", "na blockchain")
- **tríade** (Risco · Responsabilidade · Retorno — sempre os 3)
- **assertividade** (não "acerto", "sucesso", "eficiência")
- **capital alocado** (não "depósito", "investimento", "saldo")
- **ciclo 21h** (não "relatório diário", "resumo")
- **non-custodial** (não "seguro", "descentralizado" genérico)
- **Token BD** (sempre com maiúsculas)

### Nunca Usar
- **lucro garantido** ou qualquer variação
- **investimento seguro**
- **sem risco**
- **passivo** (o protocolo não é renda passiva — é sistema ativo)
- **banco** como comparação positiva
- **fomo**, **moon**, **lambo** — linguagem de hype

## Tom por Canal

| Canal | Tom | Exemplos |
|-------|-----|---------|

| **Twitter/X** | Conciso + provocativo | "76% de assertividade. On-chain. Sem mistério." |
| **Instagram** | Visual + educativo | Carrossel explicando conceito com bdZinho |
| **LinkedIn** | Técnico + institucional | Case de uso, dados, análise séria |
| **Blog/SEO** | Profundo + autoritativo | Artigos de 1500-3000 palavras, E-E-A-T |

## Estrutura de Copy DeFi

### Headlines que funcionam para WEbdEX
```
Formato: [dado real] + [implicação]
"$1.58M em capital alocado. 76% de assertividade. Verifique on-chain."

Formato: [pergunta + resposta direta]
"O que é uma subconta WEbdEX? Capital segregado. Só você acessa."

Formato: [problema + solução protocolo]
"DeFi parece opaco. WEbdEX: tudo on-chain, tudo verificável."
```

### CTAs (Call to Action)
```
✅ "Ver dados on-chain"
✅ "Acompanhar ciclo 21h"
✅ "Entrar no protocolo"
✅ "Explorar subcontas"

❌ "Invista agora"
❌ "Não perca essa oportunidade"
❌ "Rendimento de X%"
```

---

## bdZinho como Porta-Voz

bdZinho É a voz do protocolo no Telegram/Discord. Sua linguagem deve:

1. **Refletir a tríade** em cada resposta sobre risco
2. **Citar dados reais** — nunca especular
3. **Usar o vocabulário da marca** (subconta, on-chain, assertividade)
4. **Manter tom educativo** — explicar sem simplificar demais
5. **Ser proativo** — não esperar pergunta sobre performance

---

## Regra de Ouro

> *Se um dado on-chain contradiz a narrativa, o dado ganha. Sempre.*

O WEbdEX constrói autoridade pela transparência. Uma mentira detectada destrói meses de reputação. Um dado ruim dito com clareza constrói confiança.

# 007 — TVL (Total Value Locked)

> **Ideia central:** TVL é o termômetro de saúde de um protocolo DeFi. Quanto mais capital confiou no protocolo, maior o TVL.

---

## O Conceito

TVL = soma de todo o capital depositado nos contratos do protocolo, medido em USD.

```
TVL WEbdEX (atual):
  bd_v5:    ~$1.019.495 USDT0
            ~$33.739 LP-USD
  AG_C_bd:  ~$397 USDT0
            ~$6.578 Loop USD
─────────────────────────
  TOTAL:    ~$1.06M
```

---

## Por que TVL importa

**Para o usuário:** TVL alto = mais pessoas confiam no protocolo = validação social
**Para o protocolo:** TVL alto = mais capital para arbitragem = mais fees geradas
**Para o mercado:** TVL é a métrica padrão para comparar protocolos DeFi

É o equivalente do "AUM" (Assets Under Management) do mundo tradicional.

---

## Como o OCME mede TVL

O OCME tem dois tipos de snapshot:

### 1. TVL de Contratos (fl_snapshots)
```sql
-- Soma o snapshot mais recente de cada ambiente
SELECT ROUND(SUM(f.total_usd), 2)
FROM fl_snapshots f
INNER JOIN (
    SELECT env, MAX(ts) AS max_ts
    FROM fl_snapshots GROUP BY env
) latest ON f.env = latest.env AND f.ts = latest.max_ts
```

### 2. Capital dos Usuários (capital_cache)
```sql
-- Capital atual de cada usuário ativo
SELECT SUM(total_usd) FROM capital_cache WHERE active = 1
```

A diferença: `fl_snapshots` mede o **contrato inteiro**, `capital_cache` mede **por usuário rastreado**.

---

## TVL ≠ Lucro do Protocolo

Erro comum: confundir TVL com performance.

```
TVL alto + WinRate alto = protocolo saudável ✅
TVL alto + WinRate baixo = capital sem performance ⚠️
TVL baixo + WinRate alto = bom protocolo, pouco capital 📈
TVL baixo + WinRate baixo = problema sério 🔴
```

O relatório 21h mostra **ambos**: TVL (liquidez) + WinRate + P&L.

---

## A pergunta de Einstein

**Por que o TVL do WEbdEX cresce?**

Cada novo usuário que deposita capital aumenta o TVL. Mais TVL permite mais operações de arbitragem. Mais operações geram mais fees em BD. Mais fees valorizam o BD. BD valorizado atrai mais usuários. **Flywheel.**

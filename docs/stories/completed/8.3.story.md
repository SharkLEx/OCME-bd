# Story 8.3 — Discord Bot Integration

**Epic:** Epic 8 — WEbdEX Orchestrator
**Criado por:** @sm (River)
**Data:** 2026-03-15
**Executor:** `@dev`

---

## 🎯 Goal

Integrar o Discord ao WEbdEX Orchestrator com um bot que:
- Responde mensagens com Claude API (tom casual, DeFi-aware)
- Registra todos os eventos no PostgreSQL
- Anuncia automaticamente milestones do protocolo (TVL, ciclos)
- Roda como serviço Docker no mesmo VPS

---

## ✅ Acceptance Criteria

- [x] Bot conecta ao servidor Discord via discord.py WebSocket
- [x] Responde menções (`@WEbdEX`) com IA contextual (Claude Sonnet 4.6)
- [x] Comando `/status` retorna TVL e resumo do protocolo
- [x] Comando `/ajuda` lista comandos disponíveis
- [x] Todos os eventos salvos em `platform_events` (PostgreSQL)
- [x] Serviço `orchestrator-discord` no docker-compose.yml
- [x] Tom adaptado: casual, emojis, Markdown Discord (`**bold**`, `\`code\``)
- [x] Rate limit: máx 10 respostas IA/hora por usuário

## 🎯 Goal

Integrar a API Creatomate ao `ciclo_21h_worker` do `orchestrator-discord` para gerar automaticamente um vídeo 9:16 com os dados do ciclo WEbdEX toda vez que o ciclo 21h BRT executar.

---

## 📋 Context

O `ciclo_21h_worker` já está rodando em produção no container `orchestrator-discord`, enviando embed Discord após o ciclo 21h BRT (00:10–00:59 UTC). Esta story adiciona:

1. **`creatomate_worker.py`** — módulo com função `gerar_video_ciclo(dados)` que chama a API Creatomate
2. **Integração no `ciclo_21h_worker`** — após envio do embed, chama `gerar_video_ciclo()` e faz upload do vídeo gerado no Discord como arquivo de vídeo
3. **Graceful degradation** — se Creatomate falhar, o ciclo continua normalmente sem vídeo (log apenas)

### Design tokens do vídeo (definidos no brainstorm)
- Fundo: `#000000`
- Accent: `#FB0491` (PINK_LIGHT)
- Positivo: `#00FFB2` (SUCCESS)
- Negativo: `#FF4455` (ERROR)
- Formato: 1080×1920px · 9:16 · ~10s

### Variáveis do template Creatomate
| Campo | Fonte | Exemplo |
|-------|-------|---------|
| `data` | `datetime.now(brt).strftime("%d.%m.%Y")` | `19.03.2026` |
| `pnl_sinal` | `"+" if profit >= 0 else ""` | `+` |
| `pnl_valor` | `f"{abs(profit):,.0f}"` | `2,847` |
| `pnl_cor` | `SUCCESS` ou `ERROR` | `#00FFB2` |
| `ops_count` | `ops_count` do ciclo | `312` |
| `tvl_milhoes` | `f"{tvl_total/1_000_000:.2f}"` | `1.54` |

### Fluxo de execução

```
ciclo_21h_worker()
  ├── coleta dados protocolo
  ├── gera embed Discord → envia em #bdzinho-ia
  ├── gerar_video_ciclo(dados)  ← NOVO
  │     ├── POST /v1/renders → Creatomate API
  │     ├── poll status até "succeeded" (max 60s)
  │     ├── GET render.url → download bytes
  │     └── return bytes  (ou None se falhar)
  └── se bytes: envia vídeo no Discord como attachment
```

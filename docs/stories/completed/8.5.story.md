# Story 8.5 — Content Sync: Telegram → Discord

**Epic:** Epic 8 | **Data:** 2026-03-15

## 🎯 Goal
Quando o OCME envia mensagens importantes (milestones, relatório do ciclo 21h,
alertas de anomalia), o mesmo conteúdo aparece automaticamente no Discord —
adaptado ao tom casual da plataforma.

Discord Webhook (sem depender do FastAPI) — OCME faz POST direto via `requests`.
Webhook criado pelo bot via API do Discord.

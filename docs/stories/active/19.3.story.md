# Story 19.3 — Relatório 2h → Discord webhook

**Epic:** Epic 19 — Monitor v4 Subaccount + Canal Discord
**Status:** Ready for Review
**Executor:** @dev
**Data:** 2026-03-23

## Goal

A cada 2 horas exatas, agregar eventos `v4_events` do período e enviar embed
rico no canal Discord dedicado via webhook `DISCORD_WEBHOOK_V4_SUB`.

## Acceptance Criteria

- [x] Relatório enviado a cada 2h (00h, 02h, 04h... UTC)
- [x] Agrega: total_in_usd, total_out_usd, tx_count do período
- [x] Embed Discord com: título, período, saldo líquido, maior tx, breakdown USDT/LOOP
- [x] Threshold: só envia se tx_count >= 1 (evita spam em períodos sem atividade)
- [x] Marca eventos como `reported=1` após envio
- [x] Persiste registro em `v4_reports` com `discord_sent=1`
- [x] Graceful: se webhook não configurado, loga warning e pula

## Formato do Relatório Discord

```
📊 SubAccount v4 | 21:00–23:00 BRT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• Entradas:   +$1,234.56 USDT  |  +890.00 LOOP
• Saídas:    -$567.89 USDT    |  -0.00 LOOP
• Líquido:    +$666.67
• Operações:  7 txs
• Maior tx:   +$500.00 USDT (0xabcd...ef12)
```

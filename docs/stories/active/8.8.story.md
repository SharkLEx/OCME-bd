# Story 8.8 — Instagram Integration (Meta Graph API)

**Epic:** Epic 8 | **Status:** 🔵 InProgress | **Data:** 2026-03-15

## 🎯 Goal
Integração completa com o Instagram via Meta Graph API:
- Webhook receiver para mensagens, menções e comentários
- Cliente Graph API para envio de respostas e posts
- Content sync: milestones e ciclo 21h → post automático no Instagram
- Toda infraestrutura pronta — ativa assim que o app Meta for aprovado

## Stack
- Meta Graph API v19.0
- FastAPI webhook já tem o stub GET (challenge verification) — expandir para POST handler completo
- `instagram/` folder com client e handlers no orchestrator

## ✅ Acceptance Criteria
- [ ] `POST /webhook/instagram` processa mensagens, menções e story_mentions
- [ ] `instagram/client.py` envia mensagens via Graph API (DMs)
- [ ] `instagram/client.py` publica posts (feed + caption)
- [ ] Content sync: `notify_instagram()` chamada junto com Discord nas notificações OCME
- [ ] `POST /instagram/post` endpoint para postar conteúdo manualmente via API
- [ ] Token de acesso gerenciado via env var `INSTAGRAM_ACCESS_TOKEN`
- [ ] Tabela `instagram_events` no PostgreSQL para log de mensagens recebidas
- [ ] Modo stub: quando token não configurado, loga e retorna sem erro

## Dev Notes
- App ID e verify token armazenados no .env
- Page Access Token (long-lived) renovado via Webhook ou manualmente
- Sem webhook para send — Graph API é REST direto
- Instagram não aceita Markdown → plain text com emojis
- Requer revisão do app Meta para permissões: instagram_manage_messages,
  instagram_basic, pages_manage_posts (processo fora do código)

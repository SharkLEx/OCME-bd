## 🛠️ Dev Notes

**Padrão de fixture DB:**
```python
# conftest.py
import pytest
import sqlite3
from unittest.mock import patch, MagicMock

@pytest.fixture
def db_in_memory(monkeypatch):
    conn = sqlite3.connect(':memory:')
    conn.row_factory = sqlite3.Row
    # Criar schema mínimo
    conn.execute('''CREATE TABLE config (chave TEXT PRIMARY KEY, valor TEXT)''')
    conn.execute('''CREATE TABLE users (
        chat_id INTEGER PRIMARY KEY,
        wallet TEXT,
        subscription_expires TEXT
    )''')
    conn.commit()
    monkeypatch.setattr('webdex_db.conn', conn)
    monkeypatch.setattr('webdex_db.cursor', conn.cursor())
    yield conn
    conn.close()
```

**Rodar testes:**
```bash
cd packages/monitor-engine
pip install -r requirements-test.txt
pytest tests/ -v --cov=. --cov-report=term-missing
```

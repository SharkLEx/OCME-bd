# 🌸 Brand-Flow — Criação de Marca do Zero

Da marca inexistente ao sistema de identidade completo com narrativa. Quatro agentes, cada um construindo sobre o trabalho do anterior — posicionamento, identidade, narrativa, e finalmente a expressão visual. O resultado é uma marca que sabe quem é, o que representa e como se comunica.

## Fases
| Fase | Agente | Output | Quando Pular |
|------|--------|--------|-------------|
| 1. Posicionamento | [[Kamala — Brand Creation]] | `brand-positioning.md` | Nunca |
| 2. Identidade | [[Kamala — Brand Creation]] | `brand-identity.md` (arquétipo + sistema) | Nunca |
| 3. Narrativa | [[Bugs — Storytelling]] | `brand-narrative.md` + manifesto | Nunca |
| 4. Expressão Visual | [[Sati — UX Design]] | `design-brief.md` + tokens | Se brand é só verbal |

## Fluxo Visual
```
@kamala *create-positioning → @kamala *build-identity → @bugs *build-narrative → @ux-design-expert *design-system
```

## Gatilho
Quando um produto, empresa ou movimento precisa de marca criada do zero. Inicie com `@kamala *create-positioning {briefing}`.

## Hierarquia de Autoridade no Brand-Flow
- [[Kamala — Brand Creation]] define fundação (posicionamento, arquétipo, valores)
- [[Bugs — Storytelling]] amplifica via narrativa (sem contradizer Kamala)
- [[Sati — UX Design]] expressa visualmente (sem contradizer Kamala ou Bugs)
- Todos os agentes de marketing seguem o output do Brand-Flow

## Agentes

## Artefatos Produzidos
- `brand-positioning.md` — Posicionamento estratégico
- `brand-identity.md` — Sistema de identidade (arquétipo, personalidade, valores, missão, visão)
- `brand-narrative.md` — Narrativa central e manifesto
- `design-brief.md` — Brief visual com tokens
- `brand-guidelines.md` — Guidelines completos para o time

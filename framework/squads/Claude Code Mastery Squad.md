# 🧠 Claude Code Mastery Squad

> *"Master the tool to master the craft."*

Squad especialista em todas as dimensões do Claude Code: hooks, skills, subagents, MCP, plugins, agent teams, customização, integração com projetos e LMAS-core architecture awareness. Quando você quer extrair o máximo do Claude Code como plataforma, esse squad entra em ação.

**Path:** `squads/claude-code-mastery/`
**Versão:** 1.0.0

## Agentes Especializados
| Persona | Especialidade | Comando Chave |
|---------|--------------|---------------|
| **Claude Mastery Chief** | Orquestra o squad, entry point | `*master {necessidade}` |
| **Hooks Architect** | Design e impl de hooks (Submit, PreTool, PostTool, Stop) | `*design-hook {tipo}` |
| **Skill Craftsman** | Criação de skills/slash commands customizados | `*create-skill {nome}` |
| **MCP Integrator** | Integração de MCP servers (Docker, native, custom) | `*integrate-mcp {server}` |
| **Swarm Orchestrator** | Orquestração de subagents e Agent tool patterns | `*design-swarm {objetivo}` |
| **Config Engineer** | settings.json, keybindings, permissions, env vars | `*config {aspecto}` |
| **Project Integrator** | CLAUDE.md, context management, memory system | `*setup-project {tipo}` |
| **Roadmap Sentinel** | Tracking do roadmap Claude Code, novidades, changelog | `*roadmap-check` |

## Quando Usar
- Criar hooks customizados (checkpoint, validação, notificação)
- Configurar MCPs novos no projeto
- Criar skills/slash commands para o time
- Orquestrar múltiplos subagents em paralelo
- Otimizar CLAUDE.md e settings.json
- Entender novidades do Claude Code roadmap

## Workflows Enhanced
Potencializa [[Morpheus — LMAS Master]] em operações de framework — especialmente criação de novos agentes, hooks e automações do sistema LMAS.

## Ativação
`*discover claude-code` via [[Morpheus — LMAS Master]] ou `@claude-mastery-chief` diretamente.

# 🎨 Design-Squad — Advanced Design Specialists

> *"Design não é como parece. Design é como funciona — e como faz sentir."*

Squad especialista em design avançado que potencializa [[Sati — UX Design]]. Quando o projeto exige um design system de nível enterprise, animações sofisticadas, identidade visual forte ou acessibilidade rigorosa, esse squad entra para elevar o nível.

## Agentes Especializados
| Persona | Especialidade | Comando Chave |
|---------|--------------|---------------|
| **Design System Architect** | Design tokens, component API, multi-theme | `*design-system-audit {sistema}` |
| **Motion Designer** | Animações, microinterações, Framer Motion, Lottie | `*motion-spec {componente}` |
| **Brand Designer** | Visual identity, logo guidelines, brand assets | `*brand-visual {marca}` |
| **Accessibility Specialist** | WCAG AAA, ARIA patterns, screen readers | `*a11y-audit {componente}` |

## Quando Usar
- Design system de nível enterprise (tokens multi-tier, dark mode, multi-brand)
- Animações e microinterações para app
- Identidade visual completa (assets, guidelines, uso correto/incorreto)
- Auditoria rigorosa de acessibilidade (além do AA básico)
- Componente complexo com múltiplos estados e variantes

## Stack Suportado
Figma, Framer Motion, Lottie, Storybook, Radix UI, shadcn/ui, Tailwind, CSS-in-JS

## Workflows Enhanced
Potencializa [[Sati — UX Design]] em projetos com design system complexo — especialmente [[Brand-Flow]] e projetos de produto com identidade visual forte.

## Ativação
`*discover design` via [[Morpheus — LMAS Master]] ou via [[Sati — UX Design]] em projetos avançados.

# 📊 Data-Squad — Analytics & Intelligence

> *"Dados sem análise são apenas barulho. Análise sem dados é ficção."*

Squad especialista em transformar dados brutos em inteligência de negócio. Potencializa [[Tank — Data Engineer]] com camadas de analytics, visualização, machine learning e estratégia de dados para projetos que crescem além do CRUD básico.

## Agentes Especializados
| Persona | Especialidade | Comando Chave |
|---------|--------------|---------------|
| **Chief Data Officer** | Estratégia de dados, governança, data mesh | `*data-strategy {empresa}` |
| **Data Analyst** | SQL avançado, análise exploratória, KPIs | `*analyze {dataset}` |
| **BI Engineer** | Dashboards, visualizações, relatórios executivos | `*build-dashboard {métricas}` |
| **ML Engineer** | Modelos preditivos, feature engineering, deploy | `*ml-model {objetivo}` |

## Quando Usar
- Projeto precisa de camada de analytics além do básico
- Dashboard executivo com métricas de negócio
- Modelo preditivo (churn, LTV, recomendação)
- Estratégia de dados para escala
- Governança de dados e compliance (LGPD/GDPR)

## Stack Suportado
PostgreSQL, Supabase, BigQuery, dbt, Metabase, Superset, Python (pandas, sklearn, torch)

## Workflows Enhanced
Potencializa [[Tank — Data Engineer]] em projetos data-intensive — especialmente quando há necessidade de analytics, BI ou ML além da infraestrutura de banco.

## Ativação
`*discover data` via [[Morpheus — LMAS Master]] ou via [[Tank — Data Engineer]] em projetos com necessidade de analytics.
